
  ========================================================

     FreeGSTBill - Free GST Billing Software
     By DiceCodes

  ========================================================

  QUICK START (2 steps):

  1. Double-click "Install FreeGSTBill.bat"
     - This installs everything automatically
     - Wait for it to finish (1-2 minutes first time)
     - The app will OPEN AUTOMATICALLY in your browser

  2. Start creating invoices!
     - The Welcome Guide will help you set up
     - Fill in your business details (first time only)
     - Add a client and create your first invoice


  --------------------------------------------------------
  YOUR APP URL (bookmark this!):
  --------------------------------------------------------

     http://localhost:3001

  If you close the tab, just open this link in any browser.
  If the page doesn't load, start the app first (see below).


  --------------------------------------------------------
  DAILY USE:
  --------------------------------------------------------

  Start the app    -> Double-click "FreeGSTBill" on Desktop
                      OR search "FreeGSTBill" in Start Menu
                      OR double-click FreeGSTBill.vbs here

  Open the app     -> Go to http://localhost:3001 in browser
                      (or just start the app - it opens automatically)

  Stop the app     -> Double-click "Stop FreeGSTBill.bat"
                      (or just close the terminal window)


  --------------------------------------------------------
  BEST TIP: INSTALL AS DESKTOP APP
  --------------------------------------------------------

  After the app opens in your browser:
  1. Look for the blue "Install App" bar at the top
  2. Click "Install App"
  3. Done! Now FreeGSTBill opens like a normal app
     - No browser tabs to lose
     - Find it in Start Menu
     - Works even without internet


  --------------------------------------------------------
  WHERE IS MY DATA?
  --------------------------------------------------------

  Invoices (PDF)   -> "Saved Invoices" folder (auto-created)
                      Organized by: Client Name / Month / invoice.pdf

  All app data     -> "data" folder (clients, bills, settings)
                      This is YOUR data - stays on YOUR computer

  Backups          -> Use Settings > Export to save a backup file
                      Use Settings > Import to restore from backup


  --------------------------------------------------------
  TROUBLESHOOTING:
  --------------------------------------------------------

  Page doesn't load?
    -> Double-click "FreeGSTBill" on Desktop to start the server
    -> Then open http://localhost:3001

  App won't start?
    -> Run "Install FreeGSTBill.bat" again (it won't reinstall,
       just checks everything is set up correctly)
